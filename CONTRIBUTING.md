# ü§ù Contributing to Awesome Cognitively-Kind Web

Thank you for contributing to this project! Your commitment to finding high-quality, human-centric websites helps make the internet a better place.

Please ensure your suggestion strictly adheres to the **Core Philosophy** outlined in the main `README.md` before submitting a Pull Request (PR).

## üõë Submission Criteria Check (Must-Haves)

Your suggested website must pass the following checks:

### 1. Anti-Algorithm Standard

* Does the site avoid using proprietary algorithms to decide what content a user sees next (e.g., no "For You," "Trending Now," or personalized recommendation feeds)?
* Is the content delivery primarily chronological or categorized by user choice?

### 2. Human-Authored Content

* Is the content verifiably created by a human author(s)?
* Is there an explicit or clear indication that humans were used to write/curate articles, create images, or summarize information?

### 3. Cognitive Kindness

* Does the site feature **minimal distraction** (e.g., no infinite scroll, no auto-play video/audio, no intrusive ads)?
* Is the slow consumption encouraged?
* Does it avoid the use of comment sections designed to encourage anxiety or outrage?
* Does it encurage civil and positive disscussion in comments section?

## üîó How to Submit

1.  **Fork** this repository.
2.  **Edit** the `README.md` file.
3.  Add your link to the appropriate category using the established format:

    ```markdown
    * [**Website Name**](URL) - *Brief, compelling description explaining why it is Cognitively-Kind and human-generated.*
    ```

4.  **Commit** your changes with a clear, descriptive message (e.g., `Add new link: [Website Name] to [Category]`).
5.  **Create a Pull Request (PR)** against the `main` branch of this repository.

Thank you for helping us grow the Cognitively-Kind Web!
